# cardgame

## 简介:

该游戏是一个策略卡牌游戏(炉石)。玩家可以将卡牌召唤到棋盘上和电脑进行人机对战。

## 上手指南:

使用IDEA导入项目

1.创建一个新的Run Configuration

2.选择sbt Task

3.在tasks输入框中输入run

4.点击应用。

如果使用eclipse 详见该官方文档:https://www.playframework.com/documentation/2.8.x/IDE

## 使用到的框架:

Play framework
